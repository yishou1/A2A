# Trajectory Predictor

Python HTTP Service algorithm package for M06 时间序列预测.

The package includes frozen local model bundles under `models/track_threat/`:

- `st_gnn_ship_kaggle_v1`: gate-passed ship trajectory model.
- `st_gnn_aircraft_kaggle_v1_candidate`: aircraft candidate model, exported for integration testing and demonstration.

All outputs are for simulation trajectory prediction and situation-awareness ranking only.
